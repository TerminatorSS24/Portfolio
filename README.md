## 🛠 Getting Started

### 1. Clone the repository

bash
git clone https://github.com/TerminatorSS24/Portfolio.git
cd Portfolio


### 2. Install dependencies

bash
npm install
# or
yarn


### 3. Start the development server

bash
npm run dev
# or
yarn dev


### 4. Build for production

bash
npm run build
# or
yarn build
