# 🚀 Subhraneel Saha — Developer Portfolio

Welcome to the source code of my personal developer portfolio website — a modern, responsive, animated, and performance-focused site built with **React**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **Vite**.

🔗 **Live Site**: [space-of-subhraneel.vercel.app](https://space-of-subhraneel.vercel.app)

![screenshot](https://github.com/user-attachments/assets/f07d02fc-c820-417e-9162-15b2d3e9683d)

---

# 🛠️ Tech Stack

- ⚛️ **React** (Functional Components)
- 🟦 **TypeScript**
- 🌬️ **Tailwind CSS**
- 🎞️ **Framer Motion** for smooth animations
- ⚡ **Vite** for lightning-fast development
- 💅 **PostCSS**
- 🧹 **ESLint** + **Prettier**
- 📧 **EmailJS** for contact form integration

---

# ✨ Features

- 📱 Fully responsive design (mobile-first)
- 🎬 Smooth scroll-triggered animations using Framer Motion
- 🚀 Animated section transitions on navbar click & scrolling
- 🎯 Interactive hover animations for cards and buttons
- 💫 Parallax tilt effects in Skills section
- ⌨️ Hero section with typing animation
- 🧠 Projects showcase with animated project cards
- 📬 Contact form with EmailJS integration
- 📄 Resume download functionality
- 🔍 SEO-friendly meta tags and robots.txt
- ⚡ Optimized performance and clean UI/UX

# 📦 Installation & Development

## Clone the Repository

```bash
git clone https://github.com/TerminatorSS24/Portfolio.git
```

## Navigate into the Project

```bash
cd Portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

---

# ⚙️ Build for Production

```bash
npm run build
```

---

# 🚀 Deployment

This portfolio is deployed using **Vercel**.

```bash
git add .
git commit -m "Updated portfolio"
git push
```

Vercel automatically redeploys the latest version.
